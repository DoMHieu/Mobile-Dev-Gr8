# Mobile-Dev-Gr8
Khuyến khích sử dụng kotlin vì code nó ngắn gọn và dễ đọc hơn, nhưng nếu không dùng được thì cũng không sao <br>

Giao việc:<br>
1 Quản lý tiến độ <br>
3 front end (Đào Minh Hiếu + Ngọc + Hiển)<br>
2 back end (Gia Huy + Đỗ Minh Hiếu (tạm thời: Mai Hùng))<br>
1 tester + reporter + presentation (Mai Hùng)<br>

Update tiến trình dự án ngày 24/9/2025:<br>
Hiện tại thì ứng dụng về cơ bản layout đã gần hoàn thành, tính năng control nhạc cũng hoạt động bình thường, tuy nhiên thì tính năng playlist và gợi ý nhạc vẫn chưa có tiến triển do đang thiếu API. Theo đó thì tình năng search nhạc cũng vậy.<br>
Do backend vẫn đang cố gắng để tìm API thích hợp sử dụng, trong khi đó Frontend team sẽ phải tìm hiểu cách sử dụng service để nhạc có thể chạy trên ứng dụng mà không bị tắt khi chuyển fragment. Ý tưởng của tôi là cho service hoạt động ở MainActivity.kt và control nhạc ở fragment. Tuy vậy, sẽ cần rất nhiều bước để có thể hoạt động bình thường.<br>
